# Himaka Portfolio

An original personal portfolio project for **Himaka**, built with **React + Vite**.

## Design direction

- Premium dark portfolio
- Large bold typography
- Orange accent color
- Smooth section flow
- Responsive layout for desktop, tablet, and mobile
- 3D-style placeholder visuals built with CSS

## Sections

1. Header / Navbar
2. Hero
3. Services / Skills
4. About
5. Tech Logos
6. Achievements / Awards
7. Experience / Skill Marquee
8. Projects
9. Testimonials
10. Contact
11. Footer

## Project structure

```text
src/

```
